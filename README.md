# 🎵 K-Pop Re-Entry Analysis Dashboard

An interactive web-based dashboard developed to analyze historical K-Pop music chart performance using data visualization techniques. The application provides insights into chart re-entry behavior, comeback momentum, retention period, rank recovery, album performance, and fandom intensity through interactive charts and performance metrics.

## 📌 Project Overview

The K-Pop Re-Entry Analysis Dashboard helps users explore historical music chart trends through interactive visualizations. The project processes multiple analytical datasets and presents meaningful insights into long-term chart performance using an easy-to-use dashboard built with Streamlit.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Sudheerreddy999/KPOP-ReEntry-Dashboard.git
```

Move into the project directory:

```bash
cd KPOP-ReEntry-Dashboard
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
